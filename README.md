# Τυράκι 🧀

Βρες και ακύρωσε ανεπιθύμητες συνδρομές.

## Τι είναι το Τυράκι;

Σε βοηθά να:

- 🔍 Βρεις κρυφές συνδρομές που ξέχασες
- 💰 Υπολογίσεις πόσα ξοδεύεις ετησίως
- ❌ Ακυρώσεις αυτές που δεν χρειάζεσαι
- 📊 Δημιουργήσεις αναφορά με όλες τις συνδρομές σου

## Υποστηριζόμενες Τράπεζες

- Alpha Bank
- Eurobank
- Τράπεζα Πειραιώς
- Εθνική Τράπεζα (NBG)

## Πώς λειτουργεί

1. Εξάγεις CSV από την τράπεζά σου (2-3 μήνες κινήσεων)

## Τιμολόγηση

- **Δωρεάν**: Αν βρεθούν ≤5 συνδρομές
- **€3 (μία φορά)**: Αν βρεθούν >5 συνδρομές, για να δεις όλες

## Απόρρητο

- Τα δεδομένα σου δεν φεύγουν ποτέ από τον υπολογιστή σου
- Δεν αποθηκεύουμε τραπεζικές πληροφορίες
- Οι πληρωμές γίνονται μέσω ασφαλούς Stripe Checkout

## Συνεισφορά

Pull requests είναι ευπρόσδεκτα! Ειδικά για:

- Υποστήριξη περισσότερων τραπεζών
- Οδηγίες ακύρωσης για νέες υπηρεσίες
- Βελτιώσεις στο UI
